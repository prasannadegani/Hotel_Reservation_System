# HOTEL-RESERVATION-SYSTEM

Hotel Reservation System built using **Java** that allows users to book hotel rooms, manage customer details, view room availability, and generate bills efficiently.

## Overview

This application helps manage hotel room bookings and automatically performs:

* Room Booking
* Customer Management
* Room Availability Check
* Billing System

## Technologies Used

* Java
* OOP Concepts
* Collections
* Scanner
* File Handling (Optional)

## Installation

```bash
Compile:
javac Main.java

Run:
java Main
```
